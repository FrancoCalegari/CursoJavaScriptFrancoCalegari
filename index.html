Condicionales

en esta seccion vamos a ver If, Else If y Else que vendria a ser el si, sino si y sino del español


//En codigo

let edad = 25;

if (edad < 18) {
    console.log("Eres menor de edad.");
} else if (edad >= 18 && edad < 65) {
    console.log("Eres mayor de edad pero aún no estás jubilado.");
} else {
    console.log("Eres mayor de edad y estás jubilado.");
}

//En español

Si la variable edad es menor que 18, se imprimirá "Eres menor de edad."
De lo contrario, si la edad es mayor o igual a 18 y menor que 65, se imprimirá "Eres mayor de edad pero aún no estás jubilado."
Si ninguna de las condiciones anteriores es verdadera (es decir, la edad es mayor o igual a 65), se imprimirá "Eres mayor de edad y estás jubilado."
